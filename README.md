## ✨ Funcionalidades

- 🎮 **Boneco pixel art animado** com walking animation
- 💻 **Editor de código** estilo VS Code com código digitando ao vivo  
- 🌙 **Modo claro / escuro** com preferência salva no localStorage
- ✨ **Efeito glow** nos cards ao passar o mouse
- 🃏 **Efeito 3D tilt** nas skills e formações
- 📱 **Layout responsivo** para mobile, tablet e desktop
- 📄 **Download do CV** em PDF direto pelo site
- 🎬 **Ticker animado** com tecnologias e palavras-chave
- 🔔 **Toast** ao copiar e-mail

```
portfolio-wesley/
├── index.html          # Estrutura HTML principal
```

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/WesleyTelles/portfolio-wesley.git

# Entre na pasta
cd portfolio-wesley

# Abra no navegador
# Clique duas vezes no index.html
# OU use uma extensão como Live Server no VS Code
```

> ⚠️ Por usar imagens e PDF locais, recomenda-se abrir via servidor (Live Server) em vez de duplo clique.
