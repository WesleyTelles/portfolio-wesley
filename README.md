# Wesley Telles — Portfólio

Meu portfólio pessoal, desenvolvido do zero com HTML, CSS e JavaScript puro. Sem frameworks, sem dependências, só código.

Acesse: [portfoliotelles.netlify.app](https://portfoliotelles.netlify.app)

---

## Sobre o projeto

Construí esse portfólio para apresentar quem sou, o que estou aprendendo e os projetos que já desenvolvi. Quis ir além do básico e criar algo que refletisse minha personalidade, então tem um boneco pixel art animado, um editor de código ao vivo, modo escuro e alguns detalhes que tornam a experiência mais interessante.

---

## O que tem nele

- Personagem pixel art animado no hero
- Editor de código estilo VS Code com digitação ao vivo
- Alternância entre modo claro e escuro
- Efeito glow nos cards de projeto
- Efeito 3D nas seções de skills e formação
- Download do currículo em PDF
- Layout responsivo para qualquer tela
- Seções de projetos, formação, certificações e contato
